# 🎲 Dice Rolling Simulator (Python)

A simple Python program that simulates rolling a dice.
Each time the user rolls, the program generates a random number between **1 and 6**.

---

## 📌 Features

* Simulates a real dice roll
* Generates random numbers from **1 to 6**
* Allows the user to roll multiple times
* Beginner-friendly Python project

---

## 📂 Project Structure

```
dice-rolling-simulator
│
├── main.py
└── README.md
```

---

## 🛠 Requirements

* Python **3.x**
* Uses built-in **random** module

---

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/dice-rolling-simulator.git
```

2. Navigate to the folder

```
cd dice-rolling-simulator
```

3. Run the program

```
python main.py
```
